WORKFLOW ENGINE LOGIC

Submit Request

create request
status = Pending
current_step = 1


Find Approver

1. Load workflow for request type
2. Get step = current_step
3. Resolve approver
4. Create approval task


Approve Request

update approval action = Approved

if next step exists
    current_step++
else
    status = Approved


Reject Request

status = Rejected


Return Request

status = Returned
assign back to requester
