ENTERPRISE REQUEST & APPROVAL WORKFLOW SYSTEM

System Overview
---------------

This system is a workflow engine used to manage internal organization requests.

Examples of requests:
- IT Support Request
- Leave Request
- Purchase Request
- Complaint
- Document Approval

Purpose:
To provide a flexible system where employees can create requests and
have them processed through configurable approval workflows.

Basic Flow

Requester
   ↓
Workflow Steps
   ↓
Approvers
   ↓
Final Status

Main capabilities:

- Dynamic request types
- Configurable approval workflows
- File attachments
- Approval history tracking
- Dashboard reporting
