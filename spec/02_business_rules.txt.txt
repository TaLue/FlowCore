BUSINESS RULES

Create Request
--------------

User must:
- Login
- Select request type
- Fill form fields
- Upload attachments if necessary

After submitting a request:

status = Pending
current_step = 1


Approval Actions
----------------

Approver can perform the following actions:

- Approve
- Reject
- Return


Approve

If approved:
step = step + 1

If step is the last step:
status = Approved


Reject

status = Rejected
Workflow stops.


Return

status = Returned
Request goes back to requester for modification.
